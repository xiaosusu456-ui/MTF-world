# MTF World · HRT 伴侣 (v2.6.0)

专为跨性别女性（MTF）社群打造的个人综合工具与云端资源检索系统。

作者:xiaosuda
x:@xiaosusu456

## ✨ 核心特性
- 🔍 **双路并发云端搜索**：集成极速多页模式，双路线程并发出击，零延迟体验。
- 📊 **医学级 HRT 监测**：打磨至行业一线水平的指标追踪，支持 E2/T 多项数据及曲线分析。
- 🔒 **本地隐私安全锁**：4位独立隐私码，LocalStorage 本地加密存储，守护你的安全空间。
- 📐 **高阶工具箱**：内置激素单位换算器、药代动力学（半衰期）浓度模拟曲线。

## 🛠️ 技术栈
- **Frontend**: HTML5, Tailwind CSS, Lucide Icons, Chart.js, Vanilla JS
- **Backend**: Python, Flask, Flask-CORS, requests (Hosted on Vercel)
- **Database (API)**: Meilisearch (Desuwa & TransCircle)
